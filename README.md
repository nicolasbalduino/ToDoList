## Requerimentos

TodoList necessita da versão .NET 6.0 para executar. [Install here](https://dotnet.microsoft.com/en-us/download/dotnet/6.0)

Comandos para instalar dependencias.

```sh
dotnet restore
```

Comandos para executar.

```sh
dotnet build
dotnet run
```
